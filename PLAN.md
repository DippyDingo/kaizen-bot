# 🧠 «Второй Мозг» — Полный план проекта

> «Второй Мозг» — это не просто трекер. Это RPG-игра где твоя реальная жизнь = прогресс персонажа, а ИИ-компаньон помогает стать лучшей версией себя.

---

## 🏗️ Стек технологий

| Слой | Технология | Назначение |
|------|-----------|------------|
| Бот | aiogram 3.x | Основной интерфейс в Telegram |
| Бэкенд | Django + DRF | API, админка, бизнес-логика |
| БД | PostgreSQL + SQLAlchemy | Хранение всех данных |
| Миграции | Alembic | Управление схемой БД |
| Планировщик | APScheduler | Напоминания, отчёты, автоматизация |
| ИИ (еда) | Claude Vision API | Анализ фото еды → калории и БЖУ |
| ИИ (ассистент) | Claude API | Советы, анализ, поиск музыки |
| Музыка | ytmusicapi + ACRCloud | YouTube Music + распознавание напева |
| Часы | Google Fit API | Синхронизация данных с Mi Fitness |
| Жанры | MusicBrainz API | Определение жанра по исполнителю |
| Хостинг | Railway / VPS | Деплой бота и бэкенда |
| Контейнеры | Docker + docker-compose | PostgreSQL + Django + бот |
| Конфиг | Pydantic Settings + .env | Хранение токенов и секретов |

## 🖥️ Интерфейсы

### 1. 📊 Дашборд
Одно обновляемое сообщение. Показывает:
- Дата, день недели
- Персонаж с текущей экипировкой + активный сет-бонус
- Уровень, EXP, текущая локация на карте мира
- Прогресс-бары: 💧 Вода | ✅ Дейлики | 🍽️ Ккал | 🔥 Streak
- Активные дуэли и счёт
- 🎵 Текущий трек
- Сообщение от компаньона

### 2. 🌐 Telegram Web App
- 📅 Календарь с drag-n-drop задачами
- 📊 Графики: сон, вода, EXP, финансы
- 🎮 Экран героя: HP / Мана / Стамина + экипировка
- 🗺️ Карта мира с локациями
- 📦 Канбан-доска бэклога
- 🎵 Музыкальная аналитика

### 3. ⚙️ Гибкая настройка модулей
Каждый модуль можно скрыть — данные сохраняются, просто не показывается в меню и дашборде.

## 📋 Все модули

### ✅ Задачи и Дейлики
- Добавить задачу (текст, голос) с датой и приоритетом
- Приоритеты: 🔴 важно / 🟡 обычно / 🟢 когда-нибудь
- Повторяющиеся задачи, календарь и drag-n-drop в Web App
- Дейлики = "Квесты дня" в RPG-стиле

### 🍏 Здоровье
- **💧 Вода:** кнопки +150/250/500мл, прогресс-бар, напоминание
- **🍽️ Питание:** фото → ИИ считает ккал+БЖУ; текст → ИИ парсит
- **😴 Сон:** [Лёг/Проснулся], качество 1–5, данные из Google Fit
- **🏃 Тренировки:** лог + данные из Google Fit
- **💊 Лекарства:** препарат + время + напоминание + история
- **😤 Стресс/энергия:** оценка 1–5, корреляции
- **🔄 Привычки:** 30-дневные челленджи, streak-счётчик

### 📝 Дневник
- Записи: текст, фото, голосовые кружочки
- Вечерний опрос, настроение, история и поиск

### 📦 Бэклог
- Категории + свои, статус: идея / в работе / готово
- Канбан в Web App

### 📖 Книги
- Добавить книгу, автопоиск обложки (Google Books)
- Прогресс, цитаты, цель N книг/год
- ИИ: саммари + вопросы для рефлексии

### 🎨 Творчество
- Трекер времени по проектам (Godot, Aseprite, TikTok)
- Streak творчества, портфолио-архив

### 💸 Финансы
- "кофе 250₽" → ИИ категоризирует; фото чека → ИИ распознаёт
- Бюджет + прогресс-бар, экспорт в Google Sheets

### 🧠 Саморазвитие
- Трекер курсов, база знаний, Spaced Repetition
- Ежедневный вопрос от ИИ

### 🎯 Продуктивность
- Помодоро-таймер с аналитикой
- Time tracking по категориям
- Еженедельный обзор по воскресеньям

### 🎵 Музыка (YouTube Music)
- По описанию → Claude находит трек
- По напеву → ACRCloud (1000 req/день бесплатно) → fallback к Claude
- По фото обложки → Claude Vision определяет
- Статистика жанров, корреляция музыка ↔ продуктивность

|---------|-----|

| Победа в дуэли | +100 |

### Состояния персонажа

- 😐 Обычный — средние показатели
- 💪 Бодрый — норма по всему
- 🔥 Легендарный — всё идеально 7+ дней

| ✨ Аура | Огонь, Молния, Тьма, Лёд, Сакура |
| 🪶 Крылья | Демонические, Ангельские, Механические |

### Редкости

| Редкость | Шанс | Визуальный эффект |
|---------|------|-----------------|
| ⚪ Common | 55% | Без эффекта |
| 🟢 Rare | 25% | Лёгкое свечение |
| 🔵 Epic | 12% | Пульсирующий контур |
| 🟣 Legendary | 7% | Анимированные частицы |
| 🟡 Mythic | 1% | Полная анимация + звук |

**Mythic:** 1 предмет в сезон, только из Mythic-кейса или финала Battle Pass, нельзя передать.

### Тематические серии

| Серия | Стиль |
|-------|-------|
| ⚔️ Тёмное фэнтези | Мрачно, готично |
| 🌸 Аниме | Японская анимация |
| 🤖 Киберпанк | Неон, технологии |
| 🧙 Магия | Классическое фэнтези |
| 🏮 Восток | Китай, Япония, самураи |
| 🌟 Небесное | Ангелы, свет |
| 👾 Пиксель-арт | Ретро 8-бит |
| 🎭 Разное | Мемы, повседневность |

### Сет-бонусы
- 3/5 предметов серии → частицы вокруг персонажа
- 5/5 → полный эффект + уникальный титул + анимация входа

### Механики предметов
- **Разборка:** Common +10 🪙 → Legendary +200 🪙 или 80 💎
- **Крафт:** 3 одинаковых = 1 следующей редкости (до Legendary)
- **Торговля:** Common/Rare свободно, Epic 1 раз, Legendary/Mythic нельзя

## 💰 Монетизация

### Две валюты
- 🪙 **Монеты** — зарабатываются в игре, тратятся в базовом магазине
- 💎 **Кристаллы** — покупаются за деньги, тратятся на эксклюзив

### Кристаллы — пакеты
- 💎 80 — 99₽ | 💎 200 — 229₽ ⭐ | 💎 500 — 499₽ | 💎 1200 — 999₽

### 💎 Premium (299₽/мес или 2490₽/год)
- ИИ-компаньон и ассистент без ограничений
- Аналитика за всё время
- Безлимит друзей и дуэлей
- Все интеграции
- +150 💎 каждый месяц
- Золотая рамка навсегда

### 🎴 Battle Pass (299 💎/сезон)
Сезон = 1 месяц, новая тема каждый месяц.
- **Бесплатная ветка:** монеты, бейджи, Common/Rare предметы
- **Премиум ветка:** Epic (ур.5) → Legendary (ур.20) → Mythic (ур.30)

### 🎰 Кейсы

| Кейс | Цена | Содержимое |
|------|------|-----------|
| Обычный | 150 🪙 | Случайный предмет |
| Тематический | 200 🪙 / 80 💎 | Предметы одной серии |
| Слотовый | 120 🪙 | Предмет конкретного слота |
| Премиум | 150 💎 | Гарантировано Epic+ |
| Mythic | 500 💎 | Mythic предмет сезона |

**Pity:** каждые 10 без Rare → гарантированный Rare. Каждые 50 без Epic → гарантированный Epic.

### 🏪 Магазин
- Обновляется каждый понедельник
- Эксклюзив недели (7 дней!)
- Постоянные позиции: рамки, титулы, щиты streak

### 🎁 Ежедневные бонусы
- День 1–6: +50 → +150 🪙 (растёт)
- День 7: +300 🪙 + бесплатный кейс
- День 30: +500 🪙 + уникальный предмет месяца

### 🤝 Рефералы
- Друг зарегистрировался → ты +200 🪙
- Друг купил Premium → ты +500 🪙 + 50 💎
- 25 друзей → эксклюзивный Legendary предмет

## 🤖 ИИ-система

### Персональный компаньон
Выбираешь при регистрации имя и характер:
- 🧙 Мудрый наставник — спокойный, философский
- ⚔️ Суровый тренер — жёсткий, мотивирует через вызов
- 🌸 Добрый друг — мягкий, поддерживающий
- 😈 Дерзкий соперник — провоцирует, соревнуется

Компаньон помнит всю историю и говорит конкретно:
> "Никита, вчера ты выпил только 800мл. Сегодня уже 14:00, а у тебя 0мл. Всё окей?"

### Три режима ИИ
1. **📊 Аналитик** (раз в неделю) — персональный инсайт с корреляциями
2. **💬 Советник** (по запросу) — анализирует реальные данные, отвечает конкретно
3. **🎯 Планировщик** (утром) — предлагает план дня на основе истории

### 🔮 Предсказания
> "По твоей истории — в пятницу вечером ты обычно пропускаешь дейлики. Напомнить раньше?"

### 🌙 Ночной разбор дня
Каждый вечер: итог дня, что хорошо, что можно лучше, мотивационное сообщение от компаньона.
